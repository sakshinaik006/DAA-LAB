Overview

This project provides a clear and practical implementation of the Insertion Sort algorithm using Java. The application accepts a sequence of integers from standard input, processes the data using insertion sort logic, and outputs the sorted sequence in ascending order. The program is designed to be easy to understand, making it suitable for beginners who want to learn how fundamental sorting algorithms work in practice.

Insertion sort operates by dividing the array into a sorted and an unsorted section. Starting from the second element, each value is compared with elements in the sorted portion and inserted into its correct position by shifting larger elements one position to the right. This step-by-step insertion process continues until all elements are sorted.

The algorithm performs the sorting in place, meaning no additional data structures are required, which helps keep memory usage low. While insertion sort is not optimal for large datasets due to its quadratic time complexity, it performs efficiently for small or nearly sorted arrays. This implementation demonstrates core programming concepts such as loops, conditional logic, array manipulation, and user input handling in Java.

Overall, this project serves as a simple example of how classic sorting techniques can be implemented in Java and can be used as a reference for learning, practice, or small-scale applications.
